FAQ

Bagaimana cara bergabung? +

Masukkan IP server Siesta ke Multiplayer Minecraft Java maupun Bedrock.

Apakah Bedrock bisa bermain? +

Ya. Server mendukung Java dan Bedrock menggunakan Geyser.

Apakah server menggunakan economy? +

Tidak. Semua transaksi dilakukan antar pemain sehingga ekonomi benar-benar dibangun oleh komunitas.

Apakah tersedia Voice Chat? +

Ya. Pemain Java dapat menggunakan Simple Voice Chat.

Apakah tersedia sistem claim? +

Tidak. Setiap pemain menjaga dan mengamankan bangunan mereka agar tercipta kesan realistis antara konflik dan masa kejayaan atau prima suatu tim atau per individu.

Apakah PvP diperbolehkan? +

Ya, PvP diperbolehkan selama tetap mengikuti aturan server dan tidak digunakan untuk mengganggu pemain lain secara berlebihan.

Bagaimana mendapatkan Rank? +

Rank diperoleh melalui halaman Donate sebagai bentuk dukungan terhadap pengembangan server. Semua rank bersifat kosmetik dan quality of life. Tidak ada sistem Pay To Win.

Menemukan Bug? +

Silakan laporkan bug melalui Discord resmi Siesta SMP agar Staff dapat segera memperbaikinya.

Bagaimana menghubungi Staff? +

Kamu bisa bergabung ke Discord resmi Siesta SMP atau menghubungi Staff yang sedang online di dalam server.

Apakah server online 24/7? +

Ya. Server aktif selama 24 jam penuh kecuali sedang dilakukan maintenance atau update besar.

Masih belum menemukan jawaban?

Gabung ke Discord Siesta SMP. Staff kami akan dengan senang hati membantu pertanyaanmu kapan pun.
